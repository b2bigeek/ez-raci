Manage Sections
What are Sections?

Sections group related rows together for better organization. They appear as bold headers in your RACI matrix.

How to use:

  1. Create sections below (e.g., "Planning", "Execution", "Closure")
  2. When adding/editing rows, assign them to a section via the dropdown
  3. Or hover over any row and click the icon to move it

Import Instructions

Select one or more sheets to import. We'll automatically detect columns with RACI values (R, A, C, I).

  • Stakeholder columns: Columns with RACI values will be marked as stakeholder/party columns
  • Information columns: Other columns will be imported as information columns
  • First column: Will be used as the activity/task name
Tip: Click on a sheet to expand and configure header/data row settings individually.
